🛠️ Tech Stack & Requirements

To replicate the environments, code blocks, and labs presented in this book, we recommend having the following installed:

  1. Python 3.8+
  2. Apache Spark 3.2.0+ (with Java 8 or 11 configured in environment variables)
  3. Apache Airflow 2.2.0+ (Celery or LocalExecutor recommended for multi-task runs)
  4. Pandas & NumPy (pip install pandas numpy pyarrow openpyxl)
  5. PostgreSQL or equivalent target transactional database server

💡 How to Run the End-to-End Project (Chapter 47)

  1. Clone the repository and navigate to the project workspace:
    git clone https://github.com/CPdevop/SQL_DE_LEARNING.git
cd SQL_DE_LEARNING
  2. Navigate to Chapter 47 and follow the detailed validation commands to run your ETL pipeline locally or orchestrate it through Airflow scheduler!
